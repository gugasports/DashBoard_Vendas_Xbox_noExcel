# Dashboard de Vendas Xbox no Excel

## Descrição

Este projeto foi desenvolvido como parte do desafio **"Criando um Dashboard de Vendas Xbox no Excel"** da **DIO (Digital Innovation One)**.

O objetivo é transformar dados brutos em informações visuais claras e úteis, permitindo uma análise eficaz do desempenho de vendas e a tomada de decisões baseadas em dados.

A proposta é conseguir demonstrar a utilização de recursos do Microsoft Excel para análise de dados, criação de tabelas dinâmicas, gráficos e indicadores visuais, facilitando a tomada de decisão baseada em dados.

---

## Objetivos do Projeto

* Organizar dados de vendas em uma estrutura adequada para análise.
* Criar indicadores (KPIs) relevantes para o negócio.
* Desenvolver um dashboard visual e interativo.
* Aplicar recursos como:

  * Tabelas Dinâmicas
  * Gráficos Dinâmicos
  * Segmentação de Dados (Slicers)
  * Formatação Condicional
  * Fórmulas do Excel
* Facilitar a interpretação das informações através da visualização dos dados.

---

## Ferramentas Utilizadas

* Microsoft Excel (Tabelas Dinâmicas, gráficos, segmentação de dados, fórmulas e formatações condicionais), além de tentar reproduzir um visual funcional para o gestor deixando o dashboard independente de quem produziu

## Dados Utilizados

A base de dados utilizada foi disponibilizada pela DIO para fins educacionais e contém informações relacionadas às vendas de produtos Xbox.

Entre os dados analisados estão, a considerar importante para o desenvolvimento do projeto, por exemplo:

* Planos de assinatura do Xbox.
* Renovações automáticas ou não e seu valor e por quanto tempo vale a assinatura.
* Planos upsell como EA Play e Minecraft e seus respectivos valores.
* Valor total de subscrição de cada membro.

---

## Dashboard

O dashboard foi desenvolvido para apresentar informações de forma simples e objetiva, permitindo acompanhar indicadores importantes do desempenho das vendas.

Exemplos de análises disponíveis:

* Receita Total
* Quantidade de Assinaturas
* Faturamento Total por tipo de assinatura
* Comparativos por categoria
* Indicadores visuais para apoio à tomada de decisão

## Como Reproduzir

1. Faça o download ou clone este repositório.

```bash
git clone https://github.com/seu-usuario/dashboard-vendas-xbox.git
```

2. Abra o arquivo **Dashboard_Vendas_Xbox.xlsx** utilizando o Microsoft Excel.

3. Caso seja solicitado, habilite a edição e o conteúdo para utilizar todos os recursos do dashboard.

4. Utilize os filtros e segmentações para explorar os dados.
